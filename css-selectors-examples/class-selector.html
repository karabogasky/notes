In this example, we have changed the font family Helvetica (with Arial or a generic sans-serif font as a backup). The base font size is set at 16 pixels. The font color is a dark gray.

We have adjusted the font size for our h1 to 32 pixels.

We have changed the font color of p to green.

We've added a class selector to this paragraph, which changes the color to blue.

This paragraph also has a class selector.

Notice by using creating type selectors for h1 and p, we are overriding the styles set in body.

By adding a class selector to the fourth and fifth ps, we are overriding the styles we originally set for p.
