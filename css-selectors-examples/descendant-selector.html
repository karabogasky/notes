Descendant Selector

In this example, we have changed the font family Helvetica (with Arial or a generic sans-serif font as a backup). The base font size is set at 16 pixels. The font color is a dark gray.

We have adjusted the font size of <h1> to 32 pixels.

We have also created two sections in this document. These are marked up by <section>.

We applied a declaration saying any <code> tag which is a descendant of <section> should be orange.

Note that it does not matter <code> is also nested within <p> tags.
