Working with Information

Software

Operating information used by a computer. These consist of instructions dictated by code.

  • operating system
  • applications
  • scripts

Input & Output

Information provided to a computer which is then processed is input.

The results generated by a computer after it processes information is output.

Binary

The most basic 'language' a computer understands.

All instructions given to a computer are converted to ones (1) and zeros (0) which represent on-and-off electrical states which make computer processors work.

Types of Information

For a computer to be valuable, we must give it information to process. There are four types of data computers work with:

  • numeric
  • character-based
  • audio / visual
  • instructional

Numeric Information

A sequence of binary can represent numbers in decimal (base 10), the numeric system we use daily.

No. Binary
0 000
1 001
2 010
3 011
4 100
5 101

Converting binary to decimal provides us with numbers for:

  • mathematical calculations
  • a foundation for representing other information familiar to humans

Numeric Systems

Because we can convert binary (base 2) to decimal (base 10), we can also convert these binary sequences to:

  • octal (base 8)
  • hexidecimal (base 16)

Octal and hexidecimal were frequently used by mainframe programmers due to memory limitations.

It is not common to see either numeric system in use today, but they still exist in some areas of computing.

Character-Based Information

Text is the form of communication which allows us to exchange information with a computer.

We work with a character set, which maps the letterforms of a language and combinations of keystrokes to a binary sequence.

ASCII

The standard character set used by computers is known as the American Standard Code for Information Interchange.

  • 128 characters -- letters, numbers, punctuation
  • binary ranges from 0000000 to 1111111
  • includes 'control' characters

In ASCII, 1110111 1100101 1100010 represents the word "web".

ASCII Table

Character values displayed in decimal, hexidecimal, octal and binary.

Instructing Computers

When we give computers instructions through commands or programs, our sequence of human-readable characters become strings.

Combinations of these strings form instructions.

These compound instructions become programs.

The syntax (words, punctuation) we use to write programs represents a language.
