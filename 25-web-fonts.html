Web Fonts

Browser-Safe Fonts

We can change the fonts used on Web pages using the CSS property font-family. This method has limitations. The fonts selected for a site will only render if the font file is located on the user's computer.

In the late 1990s (and early 2000s) designers relied on a set of common, browser-safe fonts for use based on what was available by default through Windows and Mac operating systems, but choices were limited.

(This is, in part, why we provide generic font-family declarations in CSS, such as serif, sans-serif, monospace, cursive or fantasy.)

Web Fonts

Thanks to CSS3, we can use the @font-face property, which allows us to use custom fonts hosted on our own server or through a content-delivery network.

Sounds great, right? Well, kind of.

Not all browsers support the same font file format, which means we have to provide multiple files for one font.

Using one or two Web fonts in combination with browser-safe fonts is an acceptable, balanced practice.

Where to get Web fonts

With the right software, any font file can be transformed into a Web font. However, many typefaces predate the Web and do not include licenses to be used on the Web. (To use a font on the Web without the permission of its foundry would be akin to stealing.)

The best option is to use a hosted font service. Google Fonts provides free, open-source, hosted fonts for your Web site. Fonts.com, Font Squirrel and Typekit are also options, but may require payment to license a specific font for your site. There may also be limitations as to how the font can be used.

How to use web fonts

If you are using a hosted option, the service will provide you with code to include in your HTML or CSS file. If you opt to host your own font files, additional setup is required.

Try using web fonts

You should take the time to find a font host and practice applying using web fonts.

For now, using Google Fonts is perfectly acceptable while learning. As you create web sites in the future, pay options may provide a greater selection of fonts for your design.
