CSS Math

Converting Between Units

Fixed units — pixels — are great when creating create 'pixel-perfect' designs. What you see is what you get.

But modern designs need to be fluid and responsive for different screens and devices. Using pixels can hinder our work.

How can we convert our designs from pixels to relative measurements?

Math!

Browser Defaults

Current browsers have a default font size of 16 pixels.

What if a user adjusts these settings? What if a browser maker changes the default value?*

It is okay! Using relative units, our font size, measurements will still be proportional to the newly defined font size.

* Netscape, Internet Explorer used to have default sizes of 14px.

px to em

Formula

size in pixels / parent size in pixels = ems

Examples

12px / 16px = .75em

36px / 16px = 2.25em

48px / 16px = 3em

When using ems, the parent size is based on the immediate parent, not html.

If you are calculating the font size for an element whose parent (or ancestor) also has a font size specified, you must account for it in your design.

px to percentage

Formula

(size in pixels / parent size in pixels) * 100 = percentage

Examples

(12px / 16px) * 100 = 75%

(36px / 16px) * 100 = 225%

(48px / 16px) * 100 = 300%

When using percentage, the parent size is based on the immediate parent, not html.

If you are calculating the font size for an element whose parent (or ancestor) also has a font size specified, you must account for it in your design.

View example

px to rem

Formula

size in pixels / html size in pixels = root ems

Examples

12px / 16px = .75rem

36px / 16px = 2.25rem

48px / 16px = 3rem

When using root ems, the parent size is based on the font-size property set in the html code block.

Tips

Since you are still learning, use pixels. You will see very precisely how your designs are rendered.

You can convert fixed units to fluid ones in a later iteration of your design.

Become proficient in using fixed design practices, then work on implementing fluid design practices.

When making conversions, be sure you leave a comment in your CSS as a reminder of your calculation.

font-size: 1.875em; /* 30px / 16px */
