CSS Anatomy

CSS Blocks

This is a CSS block:

selector { property: value; }

Selectors

The selector indicates which part of our HTML we want to style

This can be:

  • a specific element (<p>)
  • an id (#preamble)
  • a class (.branches-of-gov)

Complex Selectors

We are not just limited to selecting individual elements or those with id or class.

There are ways of combining different selectors to isolate specific element in our document based on how they relate to other elements.

We'll go over these at a later date.

Declaration

The part enclosed within the curly braces ({ }) is called the declaration, which is comprised of a property and value.

selector { property: value; }

We can specify multiple declarations for each selector. We just need to separate each on with a semicolon (;). 

selector {
  property: value;
  property2: value2;
}

Properties & Values

A property and its value are separated by a colon (:).

There are hundreds! of properties we can change.

Values vary based on the property.

Examples of CSS properties

  • Color
    • color
    • background-color
  • Text
    • font-family
    • font-size
    • font-style
    • font-weight
    • text-align
    • text-decoration
    • text-transform
