Block, Inline & Inline-Block Elements

HTML elements take up space

By default (regardless of browser), HTML elements have different spatial qualities:

  • block
  • inline
  • inline-block

Block Elements

  • take up the entire width of a parent element
  • always begin on new lines
  • may contain other block or inline elements

Examples of Block Elements

  • h1 - h6
  • p
  • ul, ol and li
  • all HTML5 structural tags
  • div

Inline Elements

  • occupy only the space bound by its tags
  • do not begin with a new line
  • may only contain other inline elements

Examples of Inline Elements

  • em
  • strong
  • abbr
  • img
  • span

Inline-Block Elements

  • no elements are inline-block by default
  • we must specify this property value using CSS
  • line an inline element...
    • does not begin with a new line
  • ...with some block properties
    • can take width, height, margin and padding properties

Which is it?

If you don't know offhand, check using Chrome Developer Tools. You can determine what an element is through the visual cue, or by checking the styles.

If an elements 'display' has not been overwritten using CSS, look for the user agent stylesheet value.

Syntax

We use the display property to change whether an element appears as a block, inline or inline-block element.

img { display: block; }

What if we don't want the element to display at all?

nav { display: none; }
