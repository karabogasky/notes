Attribute Selectors

Attribute Selectors

Attribute selectors allow us to target an element with a specific attribute (and the attribute's value(s)).

A full list of attribute selectors and examples can be found at Mozilla Developers Network.

Targeting an attribute

We can target an element which includes an attribute, regardless of what the attribute's content is.

a[href] will target an a tag with the href attribute.

abbr[title] will target the abbr tag with the title attribute.

Targeting an exact value

We can also single out attribute with specific values.

img[src="photo.jpg"] will match an img tag with a src of photo.jpg.

It will only match the exact value such as: <img src="photo.jpg" />.

The above example will not match <img src="my-photo.jpg" />.

Targeting a partial value

Some attribute values may be repeated in an HTML document. We can target this pattern using *=.

img[src*="photo"] will match an img tag with a src which contains photo.

The above example will match:

<img src="photo.jpg" />

<img src="my-photo.jpg" />

Targeting the start of a value

We can look to match a pattern at the start of an attribute's value using ^=.

a[href^="http"] will match any a tag in which the href attribute begins with http.

The above example will match:

<a href="http://apple.com">Apple</a>

<a href="https://gmail.com">Gmail</a>

It will not match:

<a href="index.html">Home</a>

Targeting the end of a value

We can look to match a pattern at the end of an attribute's value using $=.

a[href$=".html"] will match any a tag in which the href attribute ends with .html.

The above example will match:

<a href="index.html">Home</a>

It will not match:

<a href="http://apple.com">Apple</a>
