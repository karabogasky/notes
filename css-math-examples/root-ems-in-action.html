Root Ems in Action

We have set the base font of html to 16 pixels.

Here is an aside. We have set the font size of the aside to 0.75 rems.

We have also set the font size of <em> to 0.75 rems.

Notice how the font size does not cascade when we use rems.

The text within <aside> is equivalent to 12 pixels.

The text within <em> which is nested within <aside> also equivalent to 12 pixels.
