Ems in Action

We have set the base font of our body to 16 pixels.

Here is an aside. We have set the font size of the aside to 0.5em.

We have also set the font size of <strong> to 0.5em.

This is code outside the aside.

Notice how the font size cascades when we use ems.

The text within <aside> is equivalent to 8 pixels.

The text within <strong> which is nested within <aside> is equivalent to 4 pixels.
