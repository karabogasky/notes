Percentage in Action

We have set the base font of our body to 16 pixels.

Here is an aside. We have set the font size of the aside to 75%.

We have also set the font size of <em> to 75%.

Notice how the font size cascades when we use percentage.

The text within <aside> is equivalent to 12 pixels.

The text within <em> which is nested within <aside> is equivalent to 9 pixels.
