id & class

Sometimes we need to single out one element or a group of elements.

  • id and class allow us to be specific
  • both are "global" attributes
    • they can be used with any element

id

  • an HTML attribute used to uniquely identify a node
  • can only be used once
  • <section id="preamble"></section>

View Example

class

  • an HTML attribute used to identify similar nodes
  • can be used multiple times
    • <section class="branches-of-gov"></section>
  • can utilize multiple values
    • values should be separated by spaces
    • <section class="branches-of-gov highlighted"></section>

Tips for using id and class

  • values should:
    • start with a letter or underscore
    • be lowercase
    • try to provide context
  • be careful about overuse
    • if you can apply styles, scripts to an element to make your idea work, make that your priority
