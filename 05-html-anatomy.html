HTML Anatomy

Elements

Elements are also known as tags.

(Most) elements have an opening and closing tag.

<element></element>

Attributes & Values

Some elements may contain additional 'settings.'

We set these through attributes and values

<element attribute="value"></element>

<p class="intro">This is a sentence.</p>

Basic Elements

Basic elements construct our page.

<html>: defines the HTML document

<head>: contains information about the document

<body>: includes content to be displayed

Semantic Elements

Semantic elements describe the content enclosed within.

<p>: a paragraph

<strong>: strong emphasis

<ol>: an ordered list

<a>: an anchor (hyperlink)

Structural Elements

New to HTML5, structural elements describe the parts of a page.

<header>: header, masthead

<nav>: navigation

<main>: main content

<footer>: footer

Generic Elements

Generic elements are solely used to group other elements for styling.

They do not provide any context, and should therefore be used sparingly.

<div>: generic 'block' element

<span>: generic 'inline' element

We will address 'block' and 'inline' elements when we cover CSS.

Self-closing elements

Some tags to not require a closing element. These include:

<img />

<input />

<link />

<meta />

These are generally semantic tags.

In HTML5, the closing slash is optional, however, it provides a visual cue when scanning HTML.
