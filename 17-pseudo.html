Pseudo-Elements & Pseudo-Classes

Pseudo-Elements

A pseudo-element is an add-on for CSS selectors which lets you style specific parts of an element.

  • :before
  • :after
  • :first-letter
  • :first-line

Pseudo-elements can be found listed among properties in the MDN: CSS Reference.

Pseudo-Elements in use

Remember how I preached ornamental items could be handled through CSS?

If you wanted to place a bullet before each paragraph...

A note about glyphs in CSS

If we want to use glyphs in CSS, we cannot use its HTML entity. It needs to be converted to a 'hex' (base 16) value.

The easiest way to do this is to look up the entity number and convert it using the Entity Conversion Calculator.

(Some entity references may include the hex value.)

Pseudo-Classes

A pseudo-class is an add-ons for CSS selectors which lets you style the state of an element.

  • :focus
  • :hover
  • :visited
  • :first-child
  • :nth-child
  • :last-child

Pseudo-classes can be found listed among properties in the MDN: CSS Reference.

Pseudo-classes in action

A link has many states. It can be inactive, hovered over, active (clicked) or visited.

If we wanted to change the hover state, our code would look like this:

a {
  color: rgb(34,124,238); /* blue */
}

a:hover {
  color: rgb(215,83,46); /* red */
}

Our LINK would appear as so.

When we want to apply a change to an element's state, we always want to set a style for the default state. In this case, the links which have not been visited are blue and lack an underline.
