Box Model Components

The box model consists of:

  • content
  • padding
  • border
  • margin

Padding

Padding is the spacing between the content and border.

We can specify padding using the following CSS properties:

  • padding-top
  • padding-right
  • padding-bottom
  • padding-left

There is also a shorthand property: padding. The value specified will be applied to all sides.

Border

A border appears between the padding and margin.

A border always requires the following components:

  • width
  • style
  • color

We can specify border using the following CSS properties:

  • border-top-width
  • border-top-style
  • border-top-color
  • border-right-width
  • border-right-style
  • border-right-color
  • border-bottom-width
  • border-bottom-style
  • border-bottom-color
  • border-left-width
  • border-left-style
  • border-left-color

Border (shorthand)

What a time suck it would be to write 12 rules to style a border!

We can cut that down to four:

  • border-top
  • border-right
  • border-bottom
  • border-left

Each property requires all three values — width, style, color — in that specific order.

There is even a more efficient shorthand property: border. The values specified will be applied to all sides.

Margin

Margin encloses content, padding and border, and separates its element from others.

We can specify padding using the following CSS properties:

  • margin-top
  • margin-right
  • margin-bottom
  • margin-left

There is also a shorthand property: margin. The value specified will be applied to all sides.
