Box Model Example

If you open up Chrome Dev Tools, you can see how the box model works using this page as an example.

In Dev tools, content appears in blue; padding appears in green; border appears in yellow (although you should be able to see the border since a width must be specified); and margin appears in orange.

The content is variable in size. The div should have 20 pixels of padding on each side, a 5-pixel gray border on each side and a 30-pixel margin on each side.
