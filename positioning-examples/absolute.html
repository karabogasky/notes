Absolute positioning

An element with its position set as absolute acts like a fixed element, but relative to the nearest positioned ancestor.

If the ancestor is static, the element will be positioned in relation to the document body.

There is a div containing the red, green and blue boxes which is set to relative. It has no x- or y-axis values, so it does not move.

The red and blue boxes are static.

The green box is absolutely positioned all the way to the right side and 50 pixels from the top of its containing element (div).

If we remove position: relative; declaration from div (try it!), the green box will appear all the way to the right side and 50 pixels from the top of the window.
