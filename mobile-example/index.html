My Mobile Site

This site is designed for both the web and mobile devices.

To view the mobile version of this site, open Chrome Developer tools. Click the mobile icon () next to "Elements." In the menu above, set the Device to "Apple iPhone 5" — this will adjust the viewport dimensions.

If you uncheck "Fit" it will keep the mobile constraints, but give you a bigger viewing area to examine the page.

You may need to refresh the page to view the mobile version.

Note the differences between the two.
